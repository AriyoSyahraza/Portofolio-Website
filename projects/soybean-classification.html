Problem Statement

Identifikasi kelas kedelai yang memiliki kemiripan visual atau karakteristik fitur dapat menjadi proses yang sulit jika dilakukan secara manual. Terlebih lagi, terdapat kelas tetua dan kelas turunan yang perlu dibedakan secara konsisten. Project ini mencoba mengevaluasi apakah pendekatan machine learning dapat membantu membedakan kelas kedelai berdasarkan skenario data yang berbeda.

Objective

▸Membangun model klasifikasi multi-class untuk membedakan empat kelas kedelai.
▸Membandingkan performa MLP dan SVM pada Dataset A sampai Dataset E.
▸Menentukan konfigurasi terbaik berdasarkan Macro F1 agar evaluasi lebih adil untuk seluruh kelas.
▸Menyajikan hasil eksperimen secara lengkap agar mudah dibaca recruiter, dosen, atau reviewer teknis.

FEATURE SET SUMMARY

Dataset A–E bukan kelas target, melainkan skenario pembentukan fitur warna. Fitur warna diekstraksi dari area biji menggunakan kanal RGB, HSV, dan intensitas (I), lalu dilengkapi fitur turunan berbasis selisih kanal seperti RG_diff, RB_diff, GB_diff, dan RGB_diff untuk menangkap dominasi relatif warna permukaan biji.

Dataset A

Mean RGBHSIV sebagai ringkasan rata-rata warna global.

Dataset B

Median RGBHSIV untuk representasi yang lebih tahan terhadap outlier.

Dataset C

Kombinasi mean dan median untuk memperkaya informasi pusat distribusi warna.

Dataset D

Median dan IQR untuk menangkap pusat sekaligus sebaran warna yang robust.

Dataset E

Mean dan standard deviation sebagai color moments untuk warna dominan dan variasinya.

Seluruh set fitur warna digabungkan dengan fitur morfometri, lalu digunakan sebagai vektor fitur pada eksperimen klasifikasi MLP dan SVM.

Workflow & Process

01
Scenario Preparation

Menyiapkan skenario Dataset A sampai Dataset E sebagai variasi percobaan untuk melihat stabilitas performa model.

02
Model Setup

Menentukan dua pendekatan model, yaitu MLP untuk pendekatan neural network dan SVM untuk pembanding berbasis margin.

03
MLP Hyperparameter Search

Menguji kombinasi hidden dimensions, learning rate, dropout, weight decay, dan batch size.

04
SVM Hyperparameter Search

Menguji variasi C, gamma, kernel, dan class weight pada setiap skenario dataset.

05
Model Evaluation

Membandingkan Accuracy, Balanced Accuracy, Macro Precision, Macro Recall, Macro F1, ROC-AUC OVR Macro, dan MCC.

06
Insight & Model Selection

Memilih konfigurasi terbaik berdasarkan Macro F1 dan melihat skenario dataset yang memberi performa paling stabil.

Experiment Tracking Matters

Honest Performance Review

Portfolio lebih kuat ketika hasil eksperimen dijelaskan apa adanya, termasuk batasan dan ruang improvement.

Model Comparison
